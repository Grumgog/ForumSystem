# ForumSystem
ForumSystem - маленький проект на PHP реализующий простую систему общения на форуме.
## Использованные технологии
1. Silex - Микрофреймворк для PHP.
2. Twig - Шаблонизатор страниц
3. UIKit - Библиотека для создания html-странц
## Описание
Проект совсем маленький:
- index.php Отвечает за маршрутизацию запросов
- DBConnector - расширяет возможности PDO и предоставляет класс DBTable для запроса к таблицам BD
Учет пользовательских записей не ведется
